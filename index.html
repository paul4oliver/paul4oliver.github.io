OpenGL 3D Scene

This challenging project taught me how to render 3D shapes, apply texture, and create lighting.  I utilized the OpenGL Mathematics (GLM) library to transform graphics in this 3D environment inspired by my first trip to Duck Donuts. A view matrix is used to simulate a camera that allows a user to move about the scene using key and mouse commands. 

C++ Data Structures

This project for CS-300 DSA: Analysis and Design taught me how and when to implement various data structures. This project demonstrates the insert, delete, and search methods for linked lists, binary search trees, and hash tables. It also demonstrates the difference in time it takes for quick sort and selection sort to execute through a vector.

System Analysis & Design

I designed a new system for a company that wanted to make its services available through a new online storefront. I compared the advantages and disadvantages of using a process and object model. I developed activity, sequence, and class diagrams to articulate the processes of the new system and created a presentation using nontechnical verbiage to explain my design.

About

My name is Paul Oliver and I created this website to showcase some of my projects from school and to practice web development. I am a fourth-year student at Southern New Hampshire University and studying for a Bachelor of Science in Computer Science with a concentration in Software Engineering. This journey was a long time coming and I have greatly enjoyed making the most of it.  The reason I chose Computer Science as my major is that it would develop the technical skills to allow me to help solve complex problems while also utilizing the customer service skills I have gained throughout my previous work experience.

At the beginning of the year I started taking advantage of the GitHub Student Developer Pack. I created this website using Bootstrap Studio and am eager to explore more of the offers  that this opportunity provides. Thanks GitHub!
