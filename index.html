Projects

The projects I completed allowed me to flex my creative thought process, practice gathering data using various techniques, and understand the significance of interactive design.  (**REPO/Link coming soon)


OpenGL-Final-3D-Scene

This was a tough project! I am grateful to have learned how to render 3D shapes, apply texture, and create lighting.  There are many possibilities when it comes to creating 3D models. This project has given me experience with the inner workings of models are made. Only Visual Studio and Paint were used to make this program.


Systems Analysis and Design

In these projects, I designed a new system for a company that wanted to make its services available online. I analyzed the current system that the company had in place and compared the advantages and disadvantages of a process and object model. I used activity, sequence, and class diagrams to articulate the processes of the new system and created a presentation using nontechnical verbiage to explain my design. 


Software Security

During the first project for this course, I analyzed a codebase to create a Vulnerability Assessment Report using static testing and manual review. I created a mitigation plan to address the Common Vulnerabilities and Exposures (CVEs) identified in through a Maven Dependency Check. In the second project, I implemented a checksum using the SHA-256 hash algorithm, generated a self-sign certificate, and refactored code to convert HTTP to HTTPS protocol.


C++ Data Structures

CS-300 DSA: Analysis and Design allowed me to study the implementation of various data structures in this project. Data structures have different  runtime/space complexities for different functions and this is important to consider when selecting the right one for the situation. 


Python Text-based Game

One of my first projects! I completed this in IT-140 Introduction to Scripting. I had a lot of fun making this text-based game and gave the game and Alice in Wonderland theme. Players type in commands to move through rooms and collect objects. Beware the Jabbawocky, though!

Creating Classes in Java

This project was an introduction to the power of  classes and my first attempt at applying the principles of object-oriented programming.


Language Integration

During this project, both Python and C++ code are used to demonstrate why it sometimes makes sense to integrate two (or more) languages.
